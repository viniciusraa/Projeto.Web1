Funções

  • Criação, publicação e sincronização de sites baseados em nuvem, simplificados.
  • integração com o controle de fonte incluindo Git e Team Foundation Server.
  • Auxílio em códigos e marcação de sintaxe pata HTML5, CSS3, JavaScripit e TypeScript.
  • Edição para jQuery, jQuery Mobile, Less e Sass.
  • Simuladores mobile.
  • Manejamento de database por MySQL, Microsoft SQL Server e SQL CE.
  • Ferramenta e emplementação de arquivos e database.
  • Implementação de hospedagem compatilhada, servidores dedicados ou Microsoft Azure.
  • Publicação de sites usando FTP, FTPS e Web Deploy.
  • Motor de busca otimizado e reclamação de performance integrados.
  • Edição remota e offline.
  • Download de sites remotos para edição local.
  • Migração de database para SQL Server Compact 4.0 database para SQL Server Express ou SQL Server.
  • Extenções por plug-ins.
